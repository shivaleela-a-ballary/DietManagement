# 🥗 Diet Plan Management System

A Java Maven-based web application that helps administrators create, manage, and organize personalized diet plans. The system provides an easy-to-use interface for maintaining meal plans, calorie information, and nutritional guidance.

---

## 📖 Overview

The Diet Plan Management System is designed to simplify the management of healthy meal plans. It enables administrators to add, update, view, and delete diet plans while maintaining nutritional information in a structured database.

---

## ✨ Features

*  Add new diet plans
*  Update existing diet plans
*  Delete diet plans
*  View all diet plans
*  Breakfast planning
*  Lunch and Dinner planning
*  Calorie tracking
*  Nutritional recommendations
*  Admin dashboard for management

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/shivaleela-a-ballary/DietManagement.git
```

### Open in IntelliJ IDEA

Import the project as a Maven project.

### Configure Database

* Install MySQL
* Create a database
* Update JDBC connection details in the project configuration.

### Build the Project

```bash
mvn clean install
```

### Run

Deploy the generated WAR file on Apache Tomcat and open the application in your browser.
